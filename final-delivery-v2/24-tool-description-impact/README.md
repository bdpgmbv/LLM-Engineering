# Tool Description Impact

## The Problem
Same tools, vague descriptions vs specific descriptions. Accuracy jumps 15-30% just by writing better tool descriptions. This is the #1 quality lever for agents.

## How to Run
```
export OPENAI_API_KEY=sk-your-key
pip install -r requirements.txt
python main.py
```
http://localhost:8000/metrics | docker-compose up --build | pytest main.py -v
