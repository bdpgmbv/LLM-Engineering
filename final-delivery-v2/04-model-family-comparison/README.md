# Model Family Comparison

## The Problem
GPT-4o costs 15x more than GPT-4o-mini. Is it 15x better? For what tasks?
Most teams use the expensive model for everything and waste thousands per month.

## What We Find Out
1. Which model wins on factual questions?
2. Which wins on reasoning/math?
3. Which wins on classification?
4. How much does each cost at scale?

## How to Run
```
export OPENAI_API_KEY=sk-your-key
pip install -r requirements.txt
python main.py
```
