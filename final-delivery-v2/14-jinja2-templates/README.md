# Jinja2 Prompt Templates

## The Problem
Writing prompts by hand for each customer is impossible at scale. Jinja2 templates let you write ONE template and fill in variables for each user. 1 template x 1000 users = 1000 unique prompts.

## How to Run
```
export OPENAI_API_KEY=sk-your-key
pip install -r requirements.txt
python main.py
```
